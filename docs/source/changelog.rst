Release Notes
#############

0.3.0
=====

:Release Date: 2023-07-04

Version 0.3 is the first public release of the project, under the terms of the `MIT license`.

Supports following Metratec rfid readers:

* HF Reader

  * DeskID Iso

  * QuasarMx

  * QuasarLR

* UHF Reader

  * DeskID UHF

  * PulsarMX

  * PulsarLR
